# Zerak Router Tool 🛡️
**Developed by Zerak Softz**

A simple and effective Android app to help network owners recover their Tenda router settings and admin credentials.

## 🚀 Main Features
*   **Download Config:** Get the `RouterCfm.cfg` file instantly.
*   **Extract Passwords:** Automatically shows the admin username and password.
*   **Easy to Use:** Designed with a clean dark interface for Android.
*   **Save to Mobile:** Files are saved directly to your Downloads folder.

## 🛠️ How to Install
1.  Download the **APK** from the [Releases](https://github.com) page.
2.  Install it on your Android phone.
3.  Go to **App Settings** and allow "Files and Media" permission.

## ⚠️ Disclaimer
This tool is for educational use and for owners to recover their own router data. Do not use it for illegal purposes.
